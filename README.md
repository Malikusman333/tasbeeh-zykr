# 🕌 Tasbeeh Zikr App

A simple, clean, and distraction-free **Tasbeeh / Zikr Counter web app** built with **React** — designed to help users remember Allah easily through digital tasbeeh counting.

---

## ✨ Features

- 🔢 Digital Tasbeeh Counter
- 🔁 Reset counter anytime
- 🧠 Simple & distraction-free UI
- 📱 Fully responsive (Mobile & Desktop)
- ⚡ Fast performance using Vite
- 🎨 Clean styling with Tailwind CSS
- 🌙 Perfect for daily Zikr & remembrance

---

## 🛠 Tech Stack

| Tech        | Description                          |
|-------------|--------------------------------------|
| React       | Front-end JavaScript library         |
| Vite        | Fast dev server & bundler            |
| TailwindCSS| Utility-first CSS framework          |
| JavaScript | App logic                            |
| HTML/CSS   | Structure & styling                  |

---


Getting Started
1️⃣ Clone the Repository
git clone https://github.com/Malikusman333/tasbeeh-zykr.git
cd tasbeeh-zykr

2️⃣ Install Dependencies
npm install

3️⃣ Run Development Server
npm run dev


App will be available at:

http://localhost:5173
